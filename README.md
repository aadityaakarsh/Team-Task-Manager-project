Team Task Manager is a complete full stack web app developed using html, css, js,react, java, springboot, jwt, postgresql with role-based access (Admin/Member). Team Task Manager is a full-stack web application that allows users to create projects, assign tasks, and track progress efficiently. It includes secure authentication with JWT and role-based access control for Admin and Member users. The application provides features like task assignment, status tracking, and a dashboard to monitor overall progress and overdue tasks. It is built using React (Vite) for the frontend, Spring Boot for the backend, and PostgreSQL as the database.

**Features**
Authentication & Security JWT-based authentication Secure login & signup system Role-based access control (Admin / Member) Admin Capabilities Create, update, and delete tasks Assign tasks to team members Manage users and roles View all tasks and progress Member Capabilities View assigned tasks Update task status Track deadlines and progress Task Management Task creation with deadlines Status tracking (Pending / In Progress / Completed) Real-time updates

**Tech Stack**

**Frontend** HTML5 CSS3 JavaScript React.js

**Backend** Java Spring Boot Spring Security JWT Authentication

**Database** PostgreSQL
